# 更新日誌 (Changelog)

最新版本在最前。打包好的 Windows 版下載見 **[Releases](https://github.com/LUCAS88TSOI/zhouzhou-voice-releases/releases)**。

## [未發佈]

### 新增
- **LLM 潤色運行時自動降級**：當前 API 服務商失效（401/403/429/網路）時，自動切換到其他已設定的服務商重試；全部失敗時彈出托盤通知，而非靜默貼出未潤色的原始口語。
- 專案開源，採 **AGPL-3.0** 授權。

## [3.7] - 2026-04-18 — 並發與穩定性強化
### 修復
- 修復 13 個並發 / 生命週期 bug（快捷鍵競態、worker 重複觸發、LLM 多輪歷史競態、關閉時背景執行緒清理）。
- 檔案轉錄潤色失敗處理強化。

## [3.6.3] - 2026-04-16 — Bug 修復
### 修復
- 畸形 `config.json` 或非字串快捷鍵不再導致啟動崩潰。
- 文件轉錄輸出防覆寫（自動附加序號）；FFmpeg 提取改用暫存檔，長檔不再佔滿記憶體。
- 快捷鍵與 Repolish 設定修改後即時生效（不需重啟）。
- 空 / 雜訊 ASR 結果正確清空上次結果；SQLite 錄音歷史加執行緒鎖；音頻播放器檔案描述符洩漏修復。

## [3.6.1] - 2026-04-09 — 文件轉錄 LLM 優化
### 新增
- 文件轉錄後可選交由 LLM 自動優化文字，輸出 `{filename}_polished.txt`，長文按句子邊界分段。
### 修復
- 文件轉錄輸出格式（SRT/TXT/JSON）正確讀取配置。

## [3.6.0] - 2026-04-09 — 架構精簡重構
### 修復
- 托盤操作不再觸發兩次；錄音歷史設定重啟後不再被重置。
### 變更
- 統一執行緒安全的 GUI 調用、共享 SQLite 連接；移除約 235 行死代碼。

## [3.5.9] - 2026-04-02 — 廣東話預設熱詞表
### 新增
- 首次安裝自動載入約 130 個廣東話熱詞 + 預設替換規則 + 糾錯範本，開箱即用。

## [3.5.8] - 2026-03-29 — 安全修復
### 修復
- 自動更新下載 URL 安全驗證、更新腳本路徑注入防護；frozen dataclass 可變 list 問題。

## [3.5.7] - 2026-03-27
### 修復
- 修復打包後 GUI 不顯示（v3.5.6 遺漏 `PySide6.QtMultimedia` 模組）。

## [3.5.6] - 2026-03-24 — 自動更新
